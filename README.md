# 일정 관리 웹 어플리케이션



## 설치  

<b>1. 프로젝트 클론</b>  
```
$ git clone "https://github.com/ksmfou98/Calendar-web-app.git"
```  

<b>2. 프로젝트 설치</b>
```
$ npm install
```    

<b>3. 프로젝트 실행</b>
```
$ npm start
```
